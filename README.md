# Zappify
# Live Link: https://zappify-nine.vercel.app/

## App Name
**Zappify** - A Modern App Marketplace Platform

## Description
Zappify is a complete app discovery and management platform where users can browse, search, install, and manage applications. Users can view detailed app information including ratings, downloads, reviews, and size. The app provides real-time search functionality, sorting options, and persistent storage of installed apps using localStorage. With a clean and responsive interface, Zappify delivers a seamless experience similar to popular app stores.

### Key Features:
- Browse and search apps with real-time filtering
- View detailed app information with rating charts
- Install and uninstall apps with persistent storage
- Sort installed apps by size (High to Low / Low to High)
- Responsive design for all devices
- Toast notifications for user actions
- Loading animations for smooth navigation

## Technologies Used

### Frontend
- **React 18** - UI library
- **React Router DOM v6** - Routing
- **Tailwind CSS** - Styling
- **DaisyUI** - UI components

### Libraries
- **Recharts** - Rating charts
- **React Toastify** - Notifications
- **React Icons** - Icons
- **React Loading Indicators** - Loading animations

### Tools
- **Vite** - Build tool
- **localStorage** - Data persistence
- **Fetch API** - Data fetching
